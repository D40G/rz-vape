# rz-vape
- Vape Script [QBCORE]

## Features
- Free Script
- Basic Config
- Low Resmon
- Item Vape

## Installation
### Manual
- Put the picture named vape in the images section of our inventory
- Download the script and put it in the `[qb]` directory.
- Add the following code to your server.cfg/resouces.cfg
```
ensure rz-vape
```
